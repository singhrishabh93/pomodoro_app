# ⏳ Pomodoro – Stay Focused, Boost Productivity

Boost productivity with timed focus sessions and breaks.  
Stay focused, avoid burnout, and achieve your goals with a simple yet powerful Pomodoro app!  
Perfect for studying, working, or managing personal projects.  

---

## ✨ Features

✅ Customizable focus and break sessions  
✅ Simple and intuitive interface  
✅ Track your progress easily  
✅ Notifications to keep you on schedule  
✅ Dark & Light theme support  
✅ Works seamlessly offline

## 🚀 Getting Started

This project is built with **Flutter**.

### To run this project locally:

1. **Clone the repository**
    ```bash
    git clone https://github.com/singhrishabh93/pomodoro_app.git
    ```
2. **Navigate to the project directory**
    ```bash
    cd pomodoro_app
    ```
3. **Install dependencies**
    ```bash
    flutter pub get
    ```
4. **Run the app**
    ```bash
    flutter run
    ```
