## ğŸ§  How It Works

The **Pomodoro Technique** is simple:  
1. Work for **25 minutes** (Focus Session)  
2. Take a **5-minute short break**  
3. After 4 sessions, take a **15-minute long break**  
4. Repeat to stay productive and avoid burnout
